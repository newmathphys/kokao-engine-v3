## 3. Voting System / Система Голосования

```
┌─────────────────────────────────────────────────────────────────────────┐
│                         Weighted Voting / Взвешенное Голосование        │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  Input: nearest = [(id, sim, etalon), ...]  # k nearest etalons        │
│                                                                         │
│  ┌─────────────────────────────────────────────────────────────────┐   │
│  │  For each etalon in nearest[:top_k]:                            │   │
│  │  ┌───────────────────────────────────────────────────────────┐ │   │
│  │  │  1. dominant_class = argmax(etalon.class_counts)          │ │   │
│  │  │  2. purity = etalon.get_purity()                          │ │   │
│  │  │  3. weight = sim × purity × etalon.activity               │ │   │
│  │  │  4. votes[dominant_class] += weight                       │ │   │
│  │  └───────────────────────────────────────────────────────────┘ │   │
│  └─────────────────────────────────────────────────────────────────┘   │
│                                                                         │
│  ┌─────────────────────────────────────────────────────────────────┐   │
│  │  Final Step / Финальный Шаг:                                    │   │
│  │  ┌───────────────────────────────────────────────────────────┐ │   │
│  │  │  probabilities = softmax(votes / temperature)             │ │   │
│  │  │  prediction = argmax(probabilities)                       │ │   │
│  │  │  confidence = probabilities[prediction]                   │ │   │
│  │  └───────────────────────────────────────────────────────────┘ │   │
│  └─────────────────────────────────────────────────────────────────┘   │
│                                                                         │
│  Output: (prediction, confidence)                                       │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```
