## 🔬 ФОРМУЛА (8) — ОСНОВА ОБУЧЕНИЯ

**Из Главы 1.2 книги Косякова:**

```math
c'_i = c_i - \frac{Δ₀ · b_i}{Σ b_i²} \quad (8)
```

**Где:**
- `c_i` — весовой коэффициент i-го фактора
- `c'_i` — обновлённый вес
- `Δ₀ = S — S_т` — общая ошибка обучения
- `b_i` — i-й фактор внешней ситуации
- `Σ b_i²` — нормализующий множитель

**Полный вывод формулы см.:** [FORMULA_8_DERIVATION.md](FORMULA_8_DERIVATION.md)

## 🧠 БИОЛОГИЧЕСКАЯ ИНТЕРПРЕТАЦИЯ

| Математический объект | Биологический аналог |
|----------------------|---------------------|
| `b_i` | Сигнал от рецептора |
| `c_i⁺` | Возбуждающий синапс |
| `c_i⁻` | Тормозящий синапс |
| `S⁺` | Сумма возбуждающих сигналов |
| `S⁻` | Сумма тормозящих сигналов |

| `Δ₀` | Сигнал ошибки от управляющего нейрона |

## 📖 КЛЮЧЕВЫЕ ПРИНЦИПЫ

### 1. Распознавание Независимо от Яркости (Глава 1.4)

```math
S = \frac{S⁺}{S⁻}
```

При изменении яркости в k раз:

```math
S' = \frac{k·S⁺}{k·S⁻} = \frac{S⁺}{S⁻} = S
```

**Реализация:** `kokao/core/etalon.py` (c⁺/c⁻ веса)

### 2. Забывание Эталонов (Глава 2.2)

Проводимость синапсов постепенно снижается:

```math
\text{activity}(t+1) = \text{activity}(t) · \exp(-1/τ)
```

**Реализация:** `kokao/core/memory.py` (адаптивное забывание)

### 3. Обучение Во Сне (Глава 2.3.6)

Когда нет внешнего восприятия, система обучается внутренним эталонам.

**Реализация:** `kokao/modules/sleep.py` (консолидация памяти)

### 4. Шкала Ценностей (Глава 4.2)

Множество удовольствий упорядочено шкалой ценностей.

**Реализация:** `kokao/modules/reward.py` (шкала ценностей)

### 5. Утомляемость (Глава 4.2)

Необходимое условие переключения целей.

**Реализация:** `kokao/modules/energy.py` (утомляемость и гибернация)
