# Вывод Формулы (8) Косякова

## Источник

**Косяков Ю.Б. "Мой мозг", Глава 1.2, 1999**

---

## Шаг 1: Сигнал Управления

```math
S = \sum_{i=1}^{n} b_i · c_i \quad (4)
```

Где:
- `b_i` — i-й фактор внешней ситуации
- `c_i` — весовой коэффициент (проводимость синапса)

---

## Шаг 2: Требуемый Сигнал

```math
S_т = \sum_{i=1}^{n} b_i · c'_i \quad (1)
```

Где:
- `c'_i` — требуемый весовой коэффициент

---

## Шаг 3: Общая Ошибка Обучения

```math
Δ₀ = S — S_т \quad (5)
```

---

## Шаг 4: Изменение Весов

```math
c'_i = c_i — Δ_i \quad (2)
```

Где:
- `Δ_i` — изменение i-го веса

---

## Шаг 5: Подстановка

Подставляем (2) в (1):

```math
\sum_{i=1}^{n} b_i · (c_i — Δ_i) = S_т \quad (3)
```

---

## Шаг 6: Учёт Ошибки

Раскрываем сумму:

```math
\sum_{i=1}^{n} b_i · c_i - \sum_{i=1}^{n} b_i · Δ_i = S_т
```

Первая часть — это S (из формулы 4):

```math
S - \sum_{i=1}^{n} b_i · Δ_i = S_т
```

Выражаем сумму изменений:

```math
\sum_{i=1}^{n} b_i · Δ_i = S - S_т = Δ₀
```

---

## Шаг 7: Нормализация

Предполагаем, что изменения пропорциональны факторам:

```math
Δ_i = k · b_i
```

Тогда:

```math
\sum_{i=1}^{n} b_i · (k · b_i) = Δ₀
```

```math
k · \sum_{i=1}^{n} b_i² = Δ₀
```

Отсюда:

```math
k = \frac{Δ₀}{\sum_{i=1}^{n} b_i²}
```

---

## Шаг 8: Группировка

Подставляем k обратно:

```math
Δ_i = \frac{Δ₀ · b_i}{\sum_{i=1}^{n} b_i²} \quad (7)
```

---

## Шаг 9: Сравнение

Возвращаемся к формуле (2):

```math
c'_i = c_i — Δ_i
```

Подставляем (7):

```math
c'_i = c_i - \frac{Δ₀ · b_i}{\sum_{i=1}^{n} b_i²}
```

---

## Шаг 10: Формула (8)

**Окончательная формула:**

```math
c'_i = c_i - \frac{Δ₀ · b_i}{\sum_{i=1}^{n} b_i²} \quad (8)
```

## Реализация в Коде

```python
# kokao/core/learning.py
def train_step(c_plus, c_minus, x, target_response, lr=0.01):
    # S = Σ b_i · c_i
    current_response = np.dot(x, c_plus) - np.dot(x, c_minus)
    
    # Δ₀ = S — S_т
    error = target_response - current_response
    
    # Σ b_i²
    norm_factor = np.sum(x ** 2) + 1e-8
    
    # (Δ₀ · b_i) / Σ b_i²
    delta = (error * x) / norm_factor
    
    # c'_i = c_i ± Δ_i
    new_c_plus = c_plus + lr * delta
    new_c_minus = c_minus - lr * delta
    
    return new_c_plus, new_c_minus
```
